## 総合評価の判定基準

| 判定 | スコア範囲 | 商用化判定 |
|------|------------|------------|
| A | 90-100 | 商用リリース可能 |
| B | 70-89 | 軽微な修正で商用化可能 |
| C | 50-69 | 重要な修正が必要 |
| D | 0-49 | 根本的な見直しが必要 |

## 深刻度とスコアの関係

| 深刻度 | 発生時の影響 |
|--------|--------------|
| Critical | 該当カテゴリのスコアが大幅減点（50%以上） |
| High | 該当カテゴリのスコアが中程度減点（30-50%） |
| Medium | 該当カテゴリのスコアが軽微減点（10-30%） |
| Low | 該当カテゴリのスコアがわずかに減点（10%未満） |

## 結果記号の意味

| 記号 | 意味 |
|------|------|
| OK | 問題なし |
| NG | 問題あり（対応必須） |
| 要確認 | 手動確認が必要 |
