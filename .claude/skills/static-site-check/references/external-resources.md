# 外部リソースチェック

外部CDN・サードパーティスクリプトの使用状況を検出・報告する。

## 検出パターン

```html
<script src="https?://
<link .* href="https?://
<iframe src="https?://
```

## リスク

- 外部CDNダウンでサイト機能停止
- サプライチェーン攻撃（外部スクリプト改ざん）
- プライバシー漏洩（外部サービスへのデータ送信）

## 危険なサービス（即時警告）

以下のドメイン・サービスを検出した場合は **警告** を出力する。

### 廃止・悪用されたCDN

| ドメイン | 理由 | 対応 |
|----------|------|------|
| `polyfill.io` | 2024年に売却後、マルウェア配信に悪用 | 即時削除、代替として cdnjs や自己ホスティング |
| `cdn.polyfill.io` | 同上 | 同上 |
| `polyfill.com` | 同上 | 同上 |

### 放棄されたプロジェクト

| ドメイン/パターン | 理由 | 対応 |
|-------------------|------|------|
| `rawgit.com` | 2019年にサービス終了 | jsdelivr または自己ホスティング |
| `rawcdn.githack.com`（非公式） | 信頼性が低い | 公式CDNまたは自己ホスティング |

### 検出時の警告例

```markdown
> **警告: 危険なサービスを検出**
>
> `polyfill.io` からのスクリプト読み込みを検出しました。
> このサービスは2024年に悪意ある第三者に売却され、マルウェア配信に使用された実績があります。
> **即時削除** を強く推奨します。
```

## チェック項目

### 1. 外部CDN

検出対象:
- jQuery、Bootstrap などのライブラリ
- フォント（Google Fonts など）
- CSS フレームワーク

報告内容:
- 読み込み元ドメイン
- SRI（Subresource Integrity）の有無
- 読み込むリソースの種類（JS/CSS/フォント）

### 2. サードパーティスクリプト

検出対象:
- アナリティクス（Google Analytics, GTM など）
- 広告スクリプト
- チャットウィジェット
- SNS 埋め込み

報告内容:
- サービス名・プロバイダ
- 読み込み方法（同期/非同期）
- 読み込み元ドメイン
- 除外ができるものがないか判定

### 3. iframe 埋め込み

検出対象:
- YouTube / Vimeo 動画
- Google Maps
- 外部フォーム
- その他の埋め込みコンテンツ

報告内容:
- 埋め込み元ドメイン
- sandbox 属性の有無
- コンテンツの種類
