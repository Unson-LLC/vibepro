## 総合評価の判定基準

| 評価 | 条件 |
|------|------|
| 商用化可 | Critical: 0件、High: 0件、Medium: 2件以下 |
| 条件付き商用化可 | Critical: 0件、High: 0件、Medium: 3件以上 または High: 1-2件 |
| 商用化不可 | Critical: 1件以上 または High: 3件以上 |
