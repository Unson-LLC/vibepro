# Product

このディレクトリは、プロジェクト全体で共通するプロダクト情報を管理します。

機能ごとの具体的な仕様は、各feature配下の `01_behavior_spec.md` と `02_functional_spec.md` に記載します。

```text
docs/features/[feature-name]/01_behavior_spec.md
docs/features/[feature-name]/02_functional_spec.md
```

## ファイル一覧

| ファイル | 目的 |
|---|---|
| `00_intent.md` | define-product 前の意図確認、対象ユーザー、成功条件、制約、作らないもの |
| `01_idea.md` | プロダクトのアイデア、実現したいこと、大きなテーマ |
| `02_problem_definition.md` | 解決したい課題、原因、優先順位 |
| `03_personas.md` | 対象ユーザー、ペルソナ、利用シーン |
| `04_user_stories.md` | ユーザー種別ごとのユーザーストーリー |
| `05_product_scope.md` | MVP、Phase 2、作らないもののスコープ |
| `06_success_metrics.md` | 成功指標、計測方法、確認頻度 |
| `07_glossary.md` | プロダクト固有の用語、避ける呼び方、関係性 |
