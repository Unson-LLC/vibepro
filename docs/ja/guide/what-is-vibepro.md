# VibeProとは

VibeProは、AI支援の開発作業に使うローカルCLIです。変更を出荷する前に必要な意図、Story、Spec、Architecture、検証証跡、レビュー結果、PR準備状態を `.vibepro/` に残します。

エンジニアリング判断を置き換えるものではありません。Senior Engineerが「何を見ればよいか」「何がまだ足りないか」を判断しやすい面を作ります。

VibeProが効く場面:

- AIエージェントの実装速度に、レビュー文脈の復元が追いつかない
- 小さな差分に見えても、実行経路やプロダクト面への影響が広い
- GitHub PR本文は簡潔にしつつ、詳細証跡はartifactに残したい
- すべての変更を重くせず、リスクに応じたGateを使いたい

基本の流れ:

1. Storyを選ぶ、または作る
2. ArchitectureとSpecの根拠を揃える
3. 実装する
4. 検証とレビュー証跡を記録する
5. `vibepro pr prepare` を実行する
6. Gate状態を確認してからPR作成またはマージへ進む
